## üí¨ Getting Help

### When You're Stuck
1. **Search Documentation:** https://docs.n8n.io
2. **Search Community:** https://community.n8n.io
3. **Ask in Discord:** https://discord.gg/n8n
4. **GitHub Issues:** For bugs/feature requests

### How to Ask Good Questions
- Describe what you're trying to achieve
- Show what you've tried
- Include error messages
- Share workflow JSON (if possible)
